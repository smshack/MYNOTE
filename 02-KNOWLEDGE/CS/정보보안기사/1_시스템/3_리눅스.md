# 📌 1. 리눅스 보안

## ✅ 리눅스 개요 및 사용

| 항목  | 설명                      |
| --- | ----------------------- |
| 리눅스 | 오픈소스 기반 UNIX 계열 OS      |

---

| ---------- | ------- |

## ✅ 핵심 구성요소

| 구성     | 설명            |

## ✅ 파일 시스템 구조

```id="fs01"
부트블록 → 슈퍼블록 → 아이노드 → 데이터블록
```

---

## ✅ 구성요소

|요소|설명|
|---|---|
|부트블록|부팅 정보|

|inode|파일 메타데이터 ⭐|
|데이터 블록|실제 데이터|

---

## ✅ inode 정보

|데이터 위치|

# 📌 10. 리눅스 방화벽 (iptables)

---

## ✅ 개념

|항목|설명|
|---|---|
|iptables|패킷 필터링 도구|

---

## ✅ 기본 체인

|체인|역할|
|---|---|
|INPUT|들어오는|
|OUTPUT|나가는|
|FORWARD|전달|

---

## ✅ 예제

```bash
iptables -A INPUT -p tcp --dport 22 -j ACCEPT
```
