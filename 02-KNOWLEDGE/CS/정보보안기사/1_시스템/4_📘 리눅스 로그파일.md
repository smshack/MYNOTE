
##### 📌 1. 현재 로그인 사용자 확인
 ✅ 주요 명령어

```bash
who
w
users
```

---

#### ✅ 비교 정리

| 명령어   | 설명            |
| ----- | ------------- |
| who   | 현재 로그인 사용자    |
| w     | 사용자 + 작업 정보 ⭐ |
| users | 사용자 목록        |

👉 시험 포인트  
➡️ **w = 가장 상세**

---

### 📌 2. 로그인 / 로그아웃 정보 확인

#### ✅ 주요 명령어

```bash
last
```

#### ✅ 특징

| 항목   | 설명            |
| ---- | ------------- |
| last | 로그인/로그아웃 기록   |
| 데이터  | /var/log/wtmp |

---

#### ✅ 추가 명령어

```bash
lastlog
```

| 설명          |
| ----------- |
| 사용자 마지막 로그인 |

---

### 📌 3. 로그인 실패 정보 확인

#### ✅ 주요 명령어

```bash
lastb
```

#### ✅ 특징

| 항목    | 설명            |
| ----- | ------------- |
| lastb | 로그인 실패 기록     |
| 파일    | /var/log/btmp |

👉 핵심  
➡️ **b = bad (실패)**

---

### 📌 4. 주요 로그 파일 위치 ⭐

| 파일                | 설명      |
| ----------------- | ------- |
| /var/log/messages | 시스템 로그  |
| /var/log/secure   | 인증 로그 ⭐ |
| /var/log/wtmp     | 로그인 기록  |
| /var/log/btmp     | 실패 기록   |
| /var/log/lastlog  | 마지막 로그인 |

---

### 📌 5. syslog 사용

#### ✅ 개념

| 항목     | 설명            |
| ------ | ------------- |
| syslog | 시스템 로그 관리 서비스 |
#### ✅ 설정 파일

```bash
/etc/rsyslog.conf
```

#### ✅ 로그 확인

```bash
tail -f /var/log/messages
```

---

#### ✅ 로그 레벨 ⭐

| 레벨      | 의미     |
| ------- | ------ |
| emerg   | 시스템 불능 |
| alert   | 즉시 조치  |
| crit    | 심각     |
| err     | 오류     |
| warning | 경고     |
| info    | 일반 정보  |
| debug   | 디버깅    |

👉 암기  
➡️ **emerg → debug (위험 → 낮음)**

---

### 📌 6. 로그 관련 추가 명령어

```bash
dmesg
journalctl
```

| 명령어        | 설명           |
| ---------- | ------------ |
| dmesg      | 커널 로그        |
| journalctl | systemd 로그 ⭐ |
