# 📘 윈도우 보안 & 인증 & 프로세스 총정리

---

### 📌 1. 윈도우 인증 시스템

## ✅ 정의

| 항목  | 설명        |
| --- | --------- |
| 인증  | 사용자 신원 확인 |
| 목적  | 시스템 접근 제어 |

👉 한 줄  
➡️ **“로그인 사용자 확인”**

---

## ✅ 인증 방식

| 방식       | 설명      |
| -------- | ------- |
| NTLM     | 구형 방식   |
| Kerberos | 기본 인증 ⭐ |

## ✅ Kerberos 핵심

| 항목    | 설명         |
| ----- | ---------- |
| 티켓 기반 | 패스워드 대신 티켓 |
| 보안성   | 높음         |
| 환경    | 도메인        |

---

### 📌 2. 윈도우 인증 프로세스 구성요소 ⭐

## ✅ 주요 구성요소

| 구성요소             | 역할        |
| ---------------- | --------- |
| Winlogon.exe     | 로그인 UI 처리 |
| LSASS.exe        | 인증 핵심 ⭐   |
| SAM              | 로컬 계정 DB  |
| Active Directory | 도메인 계정    |
| Kerberos         | 인증 프로토콜   |

---

## ✅ 인증 흐름

```id="win-auth"
로그인 입력 → Winlogon → LSASS → SAM/AD → 인증 성공
```

---

### 📌 3. 윈도우 실행 프로세스

## ✅ 부팅 흐름 ⭐

```id="boot-win"
BIOS → MBR → Bootmgr → Winload → Kernel → 로그인
```

---

## ✅ 핵심 단계

| 단계          | 설명    |
| ----------- | ----- |
| Bootmgr     | 부트 로더 |
| Winload.exe | 커널 로드 |
| Kernel      | OS 시작 |

---

## ✅ 필수 프로세스

| 프로세스         | 역할     |
| ------------ | ------ |
| System       | 커널     |
| smss.exe     | 세션 관리자 |
| csrss.exe    | 사용자 환경 |
| wininit.exe  | 초기화    |
| services.exe | 서비스 관리 |
| lsass.exe    | 인증 ⭐   |
| winlogon.exe | 로그인    |


👉 시험 포인트  
➡️ **lsass = 인증 / services = 서비스 / smss = 세션**

---

## ✅ 계정 종류

| 계정            | 설명     |
| ------------- | ------ |
| Administrator | 관리자 ⭐  |
| Guest         | 제한 계정  |
| 일반 사용자        | 기본 사용자 |

## ✅ 특징

| 항목       | 설명     |
| -------- | ------ |
| SID      | 고유 식별자 |
| 그룹 기반 권한 | 권한 관리  |

---


## ✅ 개념

|항목|설명|
|---|---|
|유니버설 그룹|도메인 전체에서 사용 가능한 그룹|

## ✅ 특징

|항목|설명|
|---|---|
|범위|전체 도메인|
|사용|다수 도메인 환경|
|포함|사용자/글로벌 그룹|

---

## ✅ 그룹 종류 비교 ⭐

|종류|범위|
|---|---|
|Local Group|로컬|
|Global Group|도메인 내|
|Universal Group|전체 도메인 ⭐|
