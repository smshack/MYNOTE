# 📘 윈도우 공유 폴더 & 레지스트리 정리

---

# 📌 1. 윈도우 공유 폴더

| 항목    | 설명                |
| ----- | ----------------- |
| 공유 폴더 | 네트워크를 통해 파일/폴더 공유 |
| 목적    | 파일 공유 및 협업        |

| 기능    | 설명    |
| ----- | ----- |
| 읽기    | 파일 조회 |
| 쓰기    | 파일 수정 |
| 전체 제어 | 모든 권한 |

---

# 📌 2. 기본 공유 폴더 ⭐


|공유|설명|
|---|---|
|C$ / D$|관리자 공유 ⭐|
|ADMIN$|Windows 디렉터리|
|IPC$|통신용 공유|

👉 특징  
➡️ **숨김 공유 ($)**

---

## 🎯 핵심

➡️ **관리자만 접근 가능 (기본적으로)**

---

# 📌 3. 공유 폴더 취약점 ⭐

## ✅ 주요 취약점

| 취약점        | 설명                    |
| ---------- | --------------------- |
| 과도한 권한     | Everyone Full Control |
| 관리자 공유 노출  | C$ 외부 접근              |
| 인증 미흡      | 비밀번호 없음               |
| 게스트 계정 활성화 | 무단 접근                 |

---

## ✅ 공격 시나리오

| 유형      | 설명          |
| ------- | ----------- |
| 랜섬웨어 확산 | 공유 폴더 통해 확산 |
| 내부 침입   | 파일 탈취       |
| 권한 상승   | 관리자 공유 악용   |

---
## ✅ 보안 대책

| 방법         | 설명       |
| ---------- | -------- |
| 공유 최소화     | 필요 시만 사용 |
| 권한 제한      | 최소 권한 원칙 |
| Guest 비활성화 | 기본 차단    |
| 방화벽 설정     | SMB 제한   |

---

# 📌 4. 레지스트리

---

## ✅ 정의

| 항목    | 설명               |
| ----- | ---------------- |
| 레지스트리 | Windows 설정 정보 DB |

👉 한 줄  
➡️ **“윈도우 설정 저장소”**

---

## ✅ 루트 키 ⭐

| 키                         | 설명       |
| ------------------------- | -------- |
| HKEY_LOCAL_MACHINE (HKLM) | 시스템 설정 ⭐ |
| HKEY_CURRENT_USER (HKCU)  | 현재 사용자   |
| HKEY_CLASSES_ROOT (HKCR)  | 파일 연결    |
| HKEY_USERS (HKU)          | 모든 사용자   |
| HKEY_CURRENT_CONFIG       | 하드웨어     |

---

## ✅ 하이브 
- 레지스트리 정보를 가지고 있는 물리적인 파일

| 파일       | 설명      |
| -------- | ------- |
| SAM      | 계정 정보   |
| SYSTEM   | 시스템 설정  |
| SOFTWARE | 프로그램 정보 |
| SECURITY | 보안 정책   |

---

# 📌 5. 주요 레지스트리 키 ⭐

---

## ✅ 시스템 정보

```text
# 컴퓨터 이름
HKLM\SYSTEM\CurrentControlSet\Control\ComputerName
```

---

## ✅ OS 정보

```text
HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion
```

---

# 📌 6. 시작 프로그램 레지스트리 ⭐

---

## ✅ 자동 실행 위치

```text
HKLM\Software\Microsoft\Windows\CurrentVersion\Run
HKCU\Software\Microsoft\Windows\CurrentVersion\Run
```

👉 의미  
➡️ 부팅 시 자동 실행

---

# 📌 7. 최근 실행 명령어 ⭐

---

## ✅ Run 기록

```text
HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\RunMRU
```

👉 의미  
➡️ 실행창 기록 저장
