# 📘 윈도우 이벤트 로그 & 웹 아티팩트 분석

---

# 📌 1. 윈도우 이벤트 로그

---

| 항목     | 설명              |
| ------ | --------------- |
| 이벤트 로그 | 시스템 및 사용자 활동 기록 |
| 목적     | 문제 분석 / 보안 감사   |

👉 한 줄  
➡️ **“시스템 활동 기록”**

---
#### 로그 종류

| 로그               | 설명       |
| ---------------- | -------- |
| System           | 시스템 관련   |
| Security         | 보안/로그인 ⭐ |
| Application      | 프로그램     |
| Setup            | 설치       |
| Forwarded Events | 전달 로그    |

---

## 🎯 핵심

➡️ **Security 로그 = 로그인/보안**

---

## ✅ 이벤트 로그 위치

```text
C:\Windows\System32\winevt\Logs
```

---

## ✅ 주요 이벤트 ID ⭐

| 이벤트 ID | 의미       |
| ------ | -------- |
| 4624   | 로그인 성공 ⭐ |
| 4625   | 로그인 실패 ⭐ |
| 4634   | 로그아웃     |
| 4672   | 관리자 권한   |
| 4688   | 프로세스 생성  |

---

👉 시험 핵심  
➡️ **4624 / 4625 무조건 암기**

---

# 📌 2. 웹 아티팩트 분석

---

## ✅ 개념

| 항목     | 설명          |
| ------ | ----------- |
| 웹 아티팩트 | 웹 사용 흔적 데이터 |
| 목적     | 사용자 행위 분석   |

---

## ✅ 분석 대상 ⭐

| 대상    | 설명     |
| ----- | ------ |
| 방문 기록 | URL    |
| 쿠키    | 사용자 정보 |
| 캐시    | 웹 파일   |
| 다운로드  | 파일 기록  |
| 검색 기록 | 키워드    |

---

## 🎯 핵심

➡️ **“사용자 웹 활동 흔적”**

---

# 📌 3. 웹 아티팩트 주요 위치

---

## ✅ 공통 위치

| 항목      | 경로              |
| ------- | --------------- |
| 사용자 폴더  | `C:\Users\사용자`  |
| AppData | `AppData\Local` |

---

# 📌 4. Internet Explorer 아티팩트 ⭐

---

## ✅ 주요 수집 데이터

| 항목         | 설명      |
| ---------- | ------- |
| History    | 방문 기록 ⭐ |
| Cookies    | 쿠키      |
| Cache      | 임시 파일   |
| Download   | 다운로드    |
| Typed URLs | 입력 URL  |

---

## ✅ 주요 경로

```text
C:\Users\사용자\AppData\Local\Microsoft\Windows\WebCache
```

---

👉 핵심  
➡️ **WebCache = IE 핵심 저장소 ⭐**

---

## ✅ Typed URL 위치

```text
HKCU\Software\Microsoft\Internet Explorer\TypedURLs
```
