# 📌 1. 컴퓨터 바이러스
> 사용자 몰래 은닉하여 정상적인 프로그램이나 사용자의 데이터를 파과하는 악성 프로그램


---

## ✅ 1) 세대별 컴퓨터 바이러스 ⭐

| 세대  | 특징                 |
| --- | ------------------ |
| 1세대 | 단순 바이러스 (고정 코드)    |
| 2세대 | 암호화 바이러스           |
| 3세대 | 은폐(스텔스)            |
| 4세대 | 다형성(Polymorphic) ⭐ |
| 5세대 | 메타모픽               |

👉 핵심  
➡️ **고정 → 암호화 → 은폐 → 변형**

| 부트 바이러스  | 부트 영역 감염   |
| 매크로 바이러스 | 문서 감염      |

| 종류    | 설명       |
| ----- | -------- |
| 덮어쓰기형 | 기존 파일 손상 |
| 기생형   | 파일에 붙음 ⭐ |

# 📘 APT 공격 (Advanced Persistent Threat)

---

# 📌 1. 개념

|항목|설명|
|---|---|
|APT|특정 대상을 노리고 **장기간 지속적으로 수행되는 공격**|
|특징|은밀 + 지속 + 표적|

👉 한 줄  
➡️ **“조용히 오래 턴다”**

---

# 📌 2. 주요 특징 ⭐

|특징|설명|
|---|---|
|표적성|특정 조직 공격|
|지속성|장기간 침투|
|은밀성|탐지 어려움|
|지능성|다양한 기법 사용|

---

# 📌 3. 공격 단계 ⭐⭐⭐

```id="apt-flow"
정찰 → 침투 → 권한획득 → 내부이동 → 정보수집 → 유출 → 은폐
```

---

## ✅ 단계별 설명

|단계|설명|
|---|---|
|정찰|대상 정보 수집|
|침투|피싱, 취약점 이용|
|권한 획득|관리자 권한 탈취|
|내부 이동|네트워크 확산|
|정보 수집|중요 데이터 수집|
|유출|외부 전송|
|은폐|흔적 제거|

---

# 📌 4. 주요 공격 기법

|기법|설명|
|---|---|
|피싱|사용자 속임|
|악성코드|백도어 설치|
|취약점 공격|시스템 침투|
|계정 탈취|권한 상승|

---

---

# 📌 5. 탐지 어려운 이유 ⭐

|이유|설명|
|---|---|
|정상 트래픽 위장|탐지 회피|
|느린 공격|장기간|
|내부 이동|내부 활동|
