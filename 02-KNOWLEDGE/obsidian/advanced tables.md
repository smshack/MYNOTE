> 표 만들기

- 표는 정보를 구조화해 시각적으로 정리하는데 유용
- 프로젝트 타입라인, 공부 계획,예산 관리 등에서 데이터를 한눈에 파악 가능
```ad-info
| 제목 | 내용 |
|------|------|
```

| 제목  | 내용  |     |     |
| --- | --- | --- | --- |
|     |     |     |     |
|     |     |     |     |
|     |     |     |     |
|     |     |     |     |
|     |     |     |     |

| 제목1  | 내용   |     |
| ---- | ---- | --- |
|      |      |     |
| 내용 > | 내용 > |     |
|      | <    | ^   |
